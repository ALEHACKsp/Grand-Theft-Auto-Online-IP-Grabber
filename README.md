# Grand Theft Auto Online IP Grabber
 A C# program that grabs ip address of players in a public lobby of GTA Online.
 
# Used Modules
* IP2Location
* Newtonsoft.Json
* PcapDotNet

# Required Drivers
> WinPcap - [Download](https://www.winpcap.org/)
